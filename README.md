# Emergency-Mobile-Robot
A mobile robot which is integrated with ESP32, line following sensors, and IMU to deliver emergency supplies with blood pressure monitor.

## Components
- 1x Arduino UNO R3
- 2x LLC 3777 - Brushed Gear Motors
- 1x L298N - Motor Driver
- 12V Power Supply

## What I did
- Using power source and a motor driver to control motors (without Arduino)
- Using Arduino to control motors
- Trying different PWM to control the speed of the motors
